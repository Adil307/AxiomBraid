# Security Policy

## Supported versions

Security fixes are provided for the latest 1.x release.

## Reporting

Do not publish a suspected vulnerability with sensitive exploitation details in a
public issue. Contact the maintainer privately through the security-reporting method
configured on the future GitHub repository. Include affected versions, reproduction
steps, impact, and any suggested mitigation.

AxiomBraid does not upload datasets. Plugins and user-provided paths execute locally and
remain part of the user's trust boundary.
