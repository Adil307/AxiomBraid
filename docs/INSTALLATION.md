# Installation

## Local development

```powershell
py -m pip install -e ".[dev]"
```

## Wheel installation

```powershell
py -m pip install axiombraid-1.0.1-py3-none-any.whl
```

## Optional chart support

```powershell
py -m pip install ".[charts]"
```

AxiomBraid 1.0 supports Python 3.10 through 3.14.
