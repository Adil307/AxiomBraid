# Research readiness

Version 1.0 provides a stable software base for future empirical evaluation, but a
research paper still requires benchmark datasets, ground-truth issue labels,
comparisons with existing tools, reproducible experiment scripts, statistical
analysis, and clearly scoped novelty claims.

Recommended study questions include:

1. How accurately does normalized text inconsistency detection identify real issues?
2. Does risk-classified cleaning reduce harmful automatic transformations?
3. How often does identifier screening reduce leakage-prone features?
4. Does beginner-oriented explanation improve task accuracy or completion time?

Report precision, recall, F1, false-positive rate, runtime, memory, and harmful-change
counts. Human studies should follow university ethics and informed-consent procedures.
