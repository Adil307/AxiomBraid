# AxiomBraid 2.0 Release Notes

AxiomBraid 2.0 adds evidence-aware confidence, a five-dimensional quality profile, controlled corruption, exact ground truth, detection evaluation, quality-response analysis, and benchmarks while preserving supported Version 1 workflows.

## Highlights

- Human-readable confidence reports
- Completeness, Uniqueness, Validity, Consistency, and Integrity scores
- Synthetic missingness, duplicates, text issues, range errors, outliers, date formats, constants, and identifiers
- Precision, recall, F1, per-detector metrics, and baseline subtraction
- Runtime, memory, and scaling benchmarks
- Expanded diagnostics, migration guidance, and API stability policy
